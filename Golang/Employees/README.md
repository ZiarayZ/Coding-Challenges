# [Google]() Question
Given a map containing employee to manager mappings, find all employees under each manager who directly or indirectly reports to him.
- For example, consider below employee-manager pairs :
    - Sample Input:
        - A->A
        - B->A
        - C->B
        - D->B
        - E->D
        - F->E
    - Sample Output:
        - A-> B, D, C, E, F
        - B-> D, C, E, F
        - C->
        - D-> E, F
        - E-> F
        - F->
