# [Adobe](https://github.com/twowaits/SDE-Interview-Questions/tree/master/Adobe) Question
- Validate if a given string is numeric.
- Return 0 / 1 ( 0 for false, 1 for true ) for this problem
- Examples:
    - "0" => true
    - " 0.1 " => true
    - "abc" => false
    - "1 a" => false
    - "2e10" => true
